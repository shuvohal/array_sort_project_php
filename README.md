# array_sort_project_php
this is practice project
1.array_push() ফাংশনটি একটি অ্যারেতে নতুন মান যোগ করতে ব্যবহৃত হয়।

2.array_shift() অ্যারের প্রথম মান মুছে ফেলে

3.array_unshift() অ্যারের শুরুতে নতুন উপাদান যোগ করে।

4.array_slice() অ্যারের একটি অংশ কেটে (copy করে) নতুন অ্যারে হিসেবে ফেরত দেয়।

(মূল অ্যারেতে কোন পরিবর্তন হয় না)

5.count() অ্যারেতে মোট কতগুলো উপাদান আছে, তা জানায়।

6.array_splice() অ্যারের নির্দিষ্ট অংশ মুছে ফেলে, চাইলে নতুন মানও যোগ করতে পারে।

7.current() অ্যারের বর্তমান পয়েন্টার যেখানে আছে, সেই উপাদানের মান রিটার্ন করে।

8.end() অ্যারের শেষ উপাদানে পয়েন্টার নিয়ে যায় এবং সেটি রিটার্ন করে।

9.next() অ্যারের পরবর্তী উপাদান রিটার্ন করে।

10.prev() পয়েন্টারকে এক ধাপ পেছনে সরায় এবং সেই উপাদান রিটার্ন করে।

11.sort() অ্যারের মানগুলোকে ascending (A → Z) ক্রমে সাজায়।

12.rsort() অ্যারের মানগুলোকে descending (Z → A) ক্রমে সাজায়।

13.asort() অ্যারের value অনুযায়ী ascending order এ সাজায়, কিন্তু key গুলো অপরিবর্তিত থাকে।

14.arsort() অ্যারের value অনুযায়ী descending order এ সাজায়, key অপরিবর্তিত থাকে।

15.ksort() অ্যারের key অনুযায়ী ascending order এ সাজায়।

16.krsort() অ্যারের key অনুযায়ী descending order এ সাজায়।
